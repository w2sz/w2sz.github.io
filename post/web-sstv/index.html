Web SSTV
Dec 15, 2023

Web SSTV aims to both encode and decode SSTV using plain JavaScript and Web Audio API. Web SSTV can be run entirely offline, and on any platform from Chromebooks to phones, so long as they support JavaScript and Web Audio. By making SSTV readily available on many platforms, we aim to create educational opportunities and introduce more people to STEM and amateur radio.

Web SSTV is currently hosted at https://ckegel.github.io/Web-SSTV/. Check out the source code at https://github.com/CKegel/Web-SSTV. Contributions are welcome.
